## 呆呆面板 Flutter App v0.0.62

### 问题修复

- 修复日志详情页在深色模式下显示白色背景的问题
- 面板日志页同样适配深色模式主题亮度

### 新增功能

- 更多页新增主题模式手动切换开关，支持浅色/深色/跟随系统三种模式

### 技术说明

- `resolveLogSurfaceTheme` 新增 `appBrightness` 参数，在无服务端配置时感知应用主题亮度
- 深色模式下日志终端默认使用 `slate950` 深色背景

### 下载说明

- **Android**: 下载 APK 文件安装使用
- **iOS**: 下载 IPA 文件，通过 AltStore 或 Sideloadly 安装
