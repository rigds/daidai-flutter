## 呆呆面板 Flutter App v0.0.63

### 新增功能

- 本地推送通知支持，App 可直接向设备发送系统通知
- 本地通知设置页面，支持通知权限请求与渠道管理
- 任务通知渠道：任务执行完成或失败时推送本地通知
- 系统通知渠道：面板系统事件和安全相关通知
- 每个渠道支持独立开关和测试通知功能
- "更多"页新增"本地通知"入口

### 技术实现

- 集成 flutter_local_notifications 库
- Android POST_NOTIFICATIONS 权限声明
- 通知服务单例模式，应用启动时自动初始化
- SharedPreferences 持久化渠道开关状态

### 下载说明

- **Android**: 下载 APK 文件安装使用
- **iOS**: 下载 IPA 文件，通过 AltStore 或 Sideloadly 安装
