# 呆呆面板 APP

呆呆面板 Flutter 客户端，用于在移动端管理面板、任务、脚本、环境变量、订阅、依赖、通知、安全设置与备份恢复。

## 当前版本

- APP：`v1.2.3`
- 适配面板：`v2.3.0`

## 本次适配重点

- 修复设置页「系统管理」区域重复显示快捷入口的问题，保留完整列表入口，页面结构更清晰。
- 优化 NAS / Nginx Proxy Manager / 公网域名反代场景下的登录错误提示，遇到 403 时会提示升级面板到 `v2.3.0` 及以上，旧版可临时补充 `config.yaml` 的 `cors.origins`。
- 登录接口返回 4xx 时会优先显示明确失败原因，不再继续进入后续加载流程后才显示笼统的网络错误。
